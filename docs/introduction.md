# 项目概述

## 1. 项目简介

Chestnut是一个基于Next.js 16和Radix UI的现代Web应用程序模板，旨在提供快速开发、易于维护的前端解决方案。该项目采用了现代化的前端技术栈和最佳实践，适合团队协作开发高质量的Web应用。

## 2. 技术栈

| 技术 | 版本 | 用途 |
|------|------|------|
| Next.js | 16 | 前端框架 |
| React | 19 | UI库 |
| Radix UI | 3.2.1 | 组件库 |
| Tailwind CSS | - | CSS框架 |
| TypeScript | 5 | 类型系统 |
| Storybook | 10.1.2 | 组件开发和文档 |
| Vitest | 4.0.14 | 单元测试 |
| Bun | - | 包管理器和运行时 |

## 3. 项目目标

- 提供现代化的前端架构模板
- 支持快速开发和原型设计
- 确保代码质量和可维护性
- 提供完整的文档体系
- 支持响应式设计和主题切换
- 提供完整的测试策略

## 4. 适用场景

- 企业级Web应用开发
- 快速原型设计和开发
- 组件库开发和文档
- 现代化前端架构学习

## 5. 项目特点

- **模块化设计**：清晰的目录结构和组件化设计
- **类型安全**：全面使用TypeScript，确保代码质量
- **主题支持**：内置亮色/暗色主题切换
- **响应式设计**：支持多种屏幕尺寸
- **混合渲染**：支持SSR和CSR
- **完整的测试体系**：单元测试、组件测试、端到端测试
- **完整的文档体系**：详细的开发指南和API文档
